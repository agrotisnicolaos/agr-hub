Your starting point

agr-launchpad

Instead of starting every project from an empty folder, you start from here — a curated, battle-tested foundation that makes Claude noticeably more capable and disciplined. It is the starting place for anything you want to build.

It gives you the freedom to create however you like — and the flexibility to grow: any asset on this hub drops straight on top when your project needs something more specialized. Clone it once, trust the folder, approve a few prompts, then just describe what you want to make.

    Why these pieces, put together this way

    Behind every piece is someone credible who has spent real time getting it right — Anthropic, Jesse Vincent, Mert Köseoğlu, Alex Newman, Upstash, and Datalayer. What the launchpad adds is the arrangement: it brings these proven tools together so they work as one, covering the whole arc of building — think → design → build → review → remember. The result is freedom with a safety net — you build however you like, quietly assisted the whole way, without ever wiring anything together or worrying about how it fits.

    The repository is built so that its assets communicate with one another whenever they need to: the planner hands off to the builder, the builder to the reviewer, the verifier looks at what was actually rendered, and memory carries it all into the next session. Together they make sure nothing falls through the cracks — and that's the whole point: bridging the gap between ideation and robust project building.

    The pieces of the puzzle

    Plugins

    • superpowers Workflow engineDisciplined workflows — brainstorm before coding, write a plan, test first, debug methodically, verify before “done.” The backbone that keeps every build rigorous. Jesse Vincent
    • skill-creator AuthoringMint your own skills. Repeating a task? It scaffolds and tests a reusable skill so Claude does it your way, every time. Anthropic
    • frontend-design InterfaceDistinctive, production-grade UI that escapes the generic “AI look” — so what you ship actually looks designed. Anthropic
    • code-review QualityReviews your changes for bugs and cleanups; /code-review ultra runs a deep multi-agent pass in the cloud before you merge. Anthropic
    • context-mode FocusKeeps large tool output out of the conversation, so Claude stays fast and sharp through long sessions. Mert Köseoğlu
    • claude-mem MemoryPersistent memory across sessions — Claude remembers your project, so you never re-explain it. Alex Newman

    Live tools (MCP servers)

    • context7 Live docsPulls current docs for any library, so Claude codes against today’s API — not stale memory. Upstash
    • GitHub Your reposConnects your repos, issues, and pull requests, so Claude works inside your real GitHub flow — via GitHub’s official hosted MCP server. GitHub
    • jupyter NotebooksA live notebook kernel — Claude runs and edits .ipynb cells in real time. Datalayer
    • chrome-devtools Eyes on the UIOptional: screenshots, console, and network from a real browser — so Claude can visually verify UI work instead of assuming. Chrome DevTools

    Companion VS Code extensions

    • markitdown Documents inConverts PDFs, Word docs, and slides into Markdown Claude can actually read — right from VS Code. BioInfo · Microsoft
    • CodeTour Guided toursPlays the step-by-step code walkthroughs the bundled code-tour skill produces. Microsoft

    Already in the repo (no install — they arrive with the clone)

    The launchpad also vendors 11 universal skills — coding standards, error handling, secure coding, API design, backend & frontend patterns, webapp testing, and more — and 3 specialist agents: architect, code-architect, and code-explorer. Every one of them is documented in the catalog below: browse the skills or meet the agents.

    Simple by design. The base stays deliberately small so it's yours to shape — swap pieces, add your own skills, tune the rulebook. And it's built to grow: every plugin and skill shared here in agr·hub is made to drop straight onto this launchpad.
