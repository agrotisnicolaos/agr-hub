How this site works

Five kinds of assets

Click any card on this site and it pops out into a bigger tile that answers four things in plain words: What is it? Why is it useful? What does it work well with? and a concrete example — plus a Download button and a link to the source it came from.

Asset type

Skill

A small file that teaches Claude a specific ability — and how you like it done. Works in Claude chat, Claude Code, or both; each card says which.

Asset type

Guide

Step-by-step instructions written for beginners — numbered steps, no jargon. The first one teaches you to install, remove, create, and change skills.

Asset type

Agent

A specialist teammate Claude Code dispatches for one focused job — system design, feature blueprints, codebase exploration — working in its own context and reporting back.

Asset type

Plugin

A bundle of skills, agents, and tools for Claude Code — installed from your terminal with a couple of /plugin commands.

Asset type

Project

A complete thing I've built, showcased with screenshots and guides — yours to download, clone, and make your own.
